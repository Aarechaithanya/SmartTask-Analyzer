# OCR + Text Summarization Web Application

This is a full-stack web application that extracts text from uploaded images using **OCR (Optical Character Recognition)** and then generates a **summary** of the extracted text using an NLP-based summarizer.  
The backend is built with **Flask**, and the frontend uses **HTML, CSS, and JavaScript**.

- Upload an image (JPG, PNG, JPEG)
- Extract text using **Tesseract OCR**

- Display OCR text + summary on the frontend
- Fully responsive and clean UI
